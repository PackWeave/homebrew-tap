# PackWeave Homebrew Tap

Official [Homebrew](https://brew.sh) tap for [PackWeave](https://github.com/PackWeave) tools.

## Install

```bash
brew tap PackWeave/tap
brew install weave
```

Or in a single command:

```bash
brew install PackWeave/tap/weave
```

## What's in this tap

| Formula | Description |
|---------|-------------|
| `weave` | MCP pack manager for AI CLIs (Claude Code, Gemini CLI, Codex CLI) |

## Updating

```bash
brew update
brew upgrade weave
```
